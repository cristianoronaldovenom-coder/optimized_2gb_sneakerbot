Setup Guide

1

Find what you want to buy

Go to pokemoncenter.com/en-ca, find the product (e.g. an Elite Trainer Box), and copy the URL from your browser's address bar.

Example URL:
https://www.pokemoncenter.com/en-ca/product/290-80538

💡 If the item is currently sold out, that's fine — the bot will monitor it and buy it the moment it comes back in stock.

2

Add your shipping address

Go to the Addresses tab and add where you want the order shipped. This is your full name, street address, city, province, postal code, and phone number.

✅ First & Last Name
✅ Street Address (e.g. 123 Main St)
✅ City, Province (e.g. ON), Postal Code (e.g. K1A 0A6)
✅ Country: CA
✅ Phone number (digits only, e.g. 6135551234)
✅ Email address
3

Add your payment info

Go to the Payment Profiles tab. You need your credit or debit card details — this is how the bot pays at checkout.

✅ Card number (the 16-digit number on the front)
✅ Cardholder name (exactly as printed on card)
✅ Expiry month (e.g. 08) and year (e.g. 2027)
✅ CVV (the 3-digit code on the back)

💡 Visa and Mastercard work best. The bot supports credit cards and debit cards with a Visa/MC logo.

4

Create a Task

Go to the Tasks tab, click + New Task, and fill in:

✅ Product URL — paste the URL you copied in Step 1
✅ Site — select "Pokemon Center Canada"
✅ Keywords — type a word from the product name to make sure you get the right version (e.g. ETB, Scarlet, Booster Bundle). Leave blank if there's only one version.
✅ Shipping Address — pick the one you added in Step 2
✅ Payment Profile — pick the card you added in Step 3
✅ Notification Email — your email so you get an order confirmation
5

Start monitoring & walk away

Back on the Tasks tab, click 📡 Monitor Stock on your task. The bot checks every 2 seconds. The moment the item is in stock, it automatically adds to cart and completes checkout — no action needed from you.

💡 Once the app is deployed (not just running in development), this works even when your computer is off.

Optional extras — you do NOT need these to buy things

Optional
2Captcha API Key
Some checkout pages show a "I'm not a robot" captcha challenge. If you enable Auto-solve captchas on a task, the bot needs a 2Captcha key to solve it automatically. Without this key, the bot will still try to checkout but may get stuck if a captcha appears.
Get one at 2captcha.com — costs about $3 USD per 1,000 solves.
Optional
Discord Webhook URL
If you use Discord, you can get a message there the instant a checkout succeeds or fails — even while your phone is in your pocket. You only need one of the two webhooks — most people use Discord.
To set it up: open Discord → right-click a channel → Edit Channel → Integrations → Create Webhook → copy the URL → paste it in Settings here.
Optional
HTTP Webhook URL
This is an alternative to Discord — it sends a notification to any website or app you specify. Only useful if you're building your own notification system. Most people skip this and use Discord instead.
Optional
Proxies
Proxies hide your real IP address from Pokemon Center's servers, which helps avoid getting blocked during monitoring. You don't need one to get started, but if the bot keeps hitting bot protection warnings, adding a residential proxy will fix it.
