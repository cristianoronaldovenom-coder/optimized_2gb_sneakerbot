2GB VPS DEPLOYMENT GUIDE

1. Update server
sudo apt update && sudo apt upgrade -y

2. Install packages
sudo apt install -y docker.io git curl wget build-essential redis-server

3. Install Node.js 22
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

4. Install PM2
sudo npm install -g pm2

5. Create swap (IMPORTANT)
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

6. Clone/upload project

7. Install dependencies
npm install

8. Start redis
sudo systemctl enable redis
sudo systemctl start redis

9. Start bot
pm2 start ecosystem.config.js

10. Save PM2
pm2 save
pm2 startup

11. Monitor RAM
htop

IMPORTANT:
- Do not run high browser concurrency on 2GB RAM
- Recommended active browsers: 1-3
- Restart server weekly for stability
- Use swap at all times
